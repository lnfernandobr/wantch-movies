# wantch 

Gerenciador de filmes wantch, busque e salve seus filmes preferidos,
desenvolvido para aprender a como usar Apollo/graphQl e Meteor
em conjunto.

Demonstração ao vivo: https://wantch-t.herokuapp.com/

## Stack usada:

 - MongoDB
 - Apollo Client
 - GraphQl
 - React
 - Redux
 - Redux-Form
 - MaterialUi
